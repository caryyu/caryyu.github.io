Centos 6 telnet安装与防火墙设置

Jul 21, 2017 2017-07-21T00:00:00+08:00 on Blogging

CentOS 6 Telnet安装与防火墙设置

Telnet安装

检查linux版本

1

cat /etc/issue

检查是否已经安装telnet

1

rpm -qa | grep telnet

安装telnet客户端(可选)

1

yum install telnet

安装telnet服务端(必须)

1

yum install telnet-server

开机自动启动

因为装好telnet服务之后,默认是不开启服务的,下面我们需要修改文件来开启服务. 把 disable=yes 更改为 disable=no

1

vi /etc/xinetd.d/telnet

重启xinetd服务

1

service xinetd restart

注意

以上安全以及配置完成了Telnet,远程一般情况下可能还是无法访问,这是因为防火墙阻止了23端口.防火墙设置请参照防火墙篇. 如果远程root登陆出现incorrect password说明不允许root远程登陆,必须要使用普通账号登陆后再进行切换root,这样是出于安全的考虑,因为Telnet这一层很容易拦截到超级管理员账号.

防火墙设置(iptable)

防火墙基本操作命令

1

service iptables [status|stop|start|restart]
  • status 防火墙状态
  • stop 停止
  • start 启动
  • restart 重启

开机自动启动防火墙

1

chkconfig iptables [off|on]

防火墙设置

VI编辑防火墙文件并加入需要开放端口的一行内容.

1
2

vi /etc/sysconfig/iptables
-A INPUT -m state --state NEW -m tcp -p tcp --dport 23 -j ACCEPT

重启生效

保持后进行重启防火墙生效

1

service iptables restart
