Centos 7 内核升级

Jul 21, 2017 2017-07-21T00:00:00+08:00 on Blogging

CentOS 7 内核升级

步骤

导入信任地址（ELRepo GPG key）

1

rpm --import https://www.elrepo.org/RPM-GPG-KEY-elrepo.org

下载内核仓库

1

rpm -Uvh http://www.elrepo.org/elrepo-release-7.0-2.el7.elrepo.noarch.rpm

安装可以索引快速安装的插件

1

yum install yum-plugin-fastestmirror

查看当前内核

1

uname -r

安装最新内核

1

yum --enablerepo=elrepo-kernel install kernel-ml

设置启动项（用最新的内核启动）

查看所有启动项

1

cat /boot/grub2/grub.cfg | grep menuentry

查看默认启动项

1

grub2-editenv list

设置默认启动项

1
2
3

# 两者选其一，可以通过索引设置也可以通过名称
grub2-set-default 0
grub2-set-default 'CentOS Linux (4.8.12-1.el7.elrepo.x86_64) 7 (Core)'

重启

1

reboot
