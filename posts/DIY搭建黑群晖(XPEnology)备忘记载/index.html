Diy搭建黑群晖(xpenology)备忘记载

May 31, 2017 2017-05-31T00:00:00+08:00 on Blogging

准备说明

进入XPEnology官网下载两个必须的文件”DSM_5.2-5644.PAT”与”XPEnoboot DS3615xs5.2-5644.5.IMG”；到Synology官网下载最新”Synology Assistant”用来辅助安装程序；然后再下载Win32diskimager用来制作镜像到U盘。

注意

这种方式安装的话最好准备一个U盘长期放在NAS那台电脑上做引导启动使用，以免拔走后无法正常引导。

步骤

利用Win32diskimager制作U盘启动

把U盘插入NAS那台主机选择 install/upgrade 启动

这个启动项在最后一个，一般在第三个，需要键盘按下到底。

在自己已有的电脑上打开Synology Assistant会自动监测到已启动的NAS

依靠指示手动安装XPEnology的PAT包

以上完毕后重启NAS从启动项的第一选项默认启动

ROOT密码获取代码

以下两种方式都试试，应该就能通过root登录了。

方式一

在利用Synology Assistant安装的时候，admin的密码就是root的密码。

方式二

安装的时候应该会有输出的，如果输出没有的话执行下面的 C 语言编写的代码获取密码；这个只会根据当天安装的时间进行获取密码，过了时间则会失效。

#include <stdlib.h>
#include <time.h>
#include <stdio.h>

void main()
{
    struct timeval tvTime;
    struct tm tmOutput;

    gettimeofday(&tvTime, 0);
    localtime_r(&(tvTime.tv_sec), &tmOutput);

    tmOutput.tm_mon += 1;
    printf("password for today is: %x%02d-%02x%02d\n\n",
        tmOutput.tm_mon, tmOutput.tm_mon, tmOutput.tm_mday,
        gcd(tmOutput.tm_mon, tmOutput.tm_mday));
}

int gcd(int a, int b)
{
    return (b?gcd(b,a%b):a);
}
