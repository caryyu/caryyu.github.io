Ubuntu 16.04 个人收藏清单

Jan 23, 2020 2020-01-23T00:00:00+08:00 on Blogging

收藏清单

  • Terminator - 分屏命令行工具
  • Hyper - 基于 XTerm 分屏命令行工具
  • Postman - Restful API 调试工具
  • IntelliJ Idea - Java 后台开发工具
  • Leanote - 蚂蚁笔记，个人笔记整理工具
  • 网易音乐 - 听音乐的好工具
  • Visual Studio Code - 前端开发工具，尤其是 NodeJS 开发推荐使用
  • Monoca 字体 - 模仿 Mac OS 字体设置；
  • Electronic Wechat - 一款基于 Electron 封装 Wechat 网页版的客户端工具；
  • Shadowsocks-qt5 - 翻墙客户端工具，需配合火狐代理插件 SwitchyOmega 使用；
  • SwitchyOmega - 火狐代理设置插件；
  • DbWrench - 数据库建模工具，物理数据库图像设计；对应 Windows 中的 PowerDesigner；此工具跨平台；
  • Shutter - 强大的截图工具，代替日常截图使用；
  • flameshot - 听说这款截图工具超级好用；
  • 搜狗输入法 - 日常打字拼音输入
