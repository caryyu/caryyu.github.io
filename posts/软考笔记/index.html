说明

  • 总时差 = 最迟开始时间 - 最早开始时间 或者 最迟结束时间 - 最早结束时间

  • 自由时差 = 紧后活动最早开始时间 - 当前活动最早结束时间
  • CPM 关键路径法 - 总时差为0的活动为关键活动，连接起来就是关键路径

三点估算

活动历时估计值 = (最乐观 + 最悲观 + 4 * 最可能) / 6 活动历时方差 = (最悲观 - 最乐观) / 6

项目成本管理

投资回收期

  • 复利法公式(总利息 = 贷款总额 * (1 + 利率)^N期)

挣值法

  • PV/BCWS 计划工作量预算费用
  • AC/ACWP 已完成工作量实际费用
  • EV/BCWP 已完成工作量预算费用
  • SV 进度偏差 = EV - PV; SV > 0 提前,SV < 0 延误；
  • CV 费用偏差 = EV - AC; CV > 0 节约,CV < 0 超支；
  • CPI 费用执行指标 = EV/AC
  • SPI 进度执行指标 = EV/PV
  • 挣值法评估曲线,横轴表示时间,纵轴表示计划工作量;

完工与未完工估算

  • EAC 完工估算 = ETC + AC 或 BAC / CPI
  • ETC 未完工估算 = (BAC - EV) / CPI; ETC = EAC - AC
  • BAC 完工预算或称总预算 = BAC = PV

EAC二种情况

  • (典型)原来估算不适合以后情况： EAC = AC + ETC 或 BAC / CPI ETC = (BAC - EV) / CPI
  • (非典型)目前偏差是特列，以后不再发生类似：EAC = AC + (BAC - EV) ETC = BAC - EV

沟通管理

沟通渠道数 = n(n - 1) / 2 沟通渠道数量级 = n^2
